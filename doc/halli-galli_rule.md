# 할리갈리 구현 규칙

## 기본 구성

- 플레이 인원: 2~6명
- 덱: 딸기, 바나나, 라임, 자두 각 14장으로 총 56장
- 카드 숫자 분포(과일별): 1개 카드 5장, 2개 카드 3장, 3개 카드 3장, 4개 카드 2장, 5개 카드 1장

## 진행

1. 서버가 전체 카드를 가능한 한 같은 수로 나눠 각 플레이어의 비공개 덱에 둡니다.
2. 현재 차례인 플레이어는 자기 덱 맨 위 카드 한 장을 공개 더미에 놓습니다.
3. 모든 플레이어의 공개 더미에서 맨 위 카드만 과일별로 합산합니다.
4. 어느 한 과일이 정확히 5개면 차례와 관계없이 누구나 종을 누를 수 있습니다.

## 종 판정

- 서버가 받은 요청 중 현재 `boardRevision`과 일치하는 첫 유효 요청을 판정합니다.
- 정답이면 종을 누른 플레이어가 모든 공개 더미를 자기 덱 아래에 넣고 새 라운드를 시작합니다.
- 오답이면 종을 누른 플레이어가 다른 참가자에게 카드 한 장씩 나눠 줍니다.
- 같은 보드 상태에서 같은 플레이어가 오답 종을 반복할 수 없습니다.
- 카드가 없는 플레이어도 현재 공개 더미의 종 판정이 끝날 때까지 종을 누를 수 있습니다.

## 종료

- 공개 더미 정리 뒤 카드가 없는 플레이어는 탈락합니다.
- 두 명만 남으면 최종 라운드가 되며 다음 종 판정으로 게임을 종료합니다.
- 최종 라운드에서 정답 종은 누른 플레이어가, 오답 종은 상대 플레이어가 공개 더미를 가져갑니다.
- 종료 시 보유 카드 수로 순위를 정하고, 동률이면 마지막 유효 종의 승자를 우선합니다.
