# 다빈치 코드 규칙

## 적용 범위

- 2~4명이 플레이한다.
- 검정과 흰색 각각 0~11 숫자 타일, 총 24개를 사용한다.
- 대시 타일, 턴 타이머와 토너먼트 점수는 첫 버전에서 제외한다.
- 2~3명은 4개, 4명은 3개의 시작 타일을 받는다.
- 시작 타일과 매 턴 가져올 타일은 색상을 선택하고, 해당 색상의 남은 타일 중 하나를 무작위로 받는다.

## 코드 배열

- 숫자가 작은 타일부터 왼쪽에 놓는다.
- 숫자가 같으면 검정 타일을 흰색 타일보다 왼쪽에 놓는다.
- 공개된 타일도 원래 위치를 유지한다.

## 턴

1. 풀이 남아 있으면 검정 또는 흰색 타일 하나를 가져온다.
2. 다른 생존 플레이어의 숨겨진 타일 하나와 숫자 하나를 지목한다.
3. 정답이면 그 타일을 공개하고, 다시 추측하거나 턴을 끝낸다.
4. 오답이면 이번 턴에 가져온 타일을 공개한 채 자기 코드에 넣고 턴을 끝낸다.
5. 정답 후 스스로 턴을 끝내면 가져온 타일을 숨긴 채 자기 코드에 넣는다.
6. 풀이 비면 타일을 가져오지 않고 추측만 진행한다. 오답 또는 중단 시 바로 다음 턴으로 넘어간다.

## 탈락과 승리

- 코드의 모든 타일이 공개된 플레이어는 탈락한다.
- 탈락한 플레이어는 타일을 가져오거나 추측할 수 없다.
- 비공개 타일이 남은 마지막 플레이어가 승리한다.

## 정보 공개 원칙

- 모든 플레이어는 각 코드의 타일 순서, 색상과 공개 여부를 본다.
- 공개된 타일의 숫자만 전체 상태에 포함한다.
- 자신의 전체 숫자와 이번 턴에 가져온 타일 숫자는 개인 메시지로만 받는다.
- 숨겨진 숫자와 풀 순서는 공개 Schema, 채팅, 오류와 로그에 포함하지 않는다.
